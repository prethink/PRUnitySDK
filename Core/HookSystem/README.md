# HookSystem

HookSystem — это система хуков, предназначенная для последовательной обработки событий с возможностью их модификации, отмены или завершения.

В отличие от EventBus, который просто уведомляет подписчиков, HookSystem:
- вызывает слушателей строго по порядку (Order);
- ожидает завершения обработки каждым слушателем;
- позволяет изменить аргументы события;
- позволяет прервать дальнейшую обработку;
- возвращает итоговое состояние события.
